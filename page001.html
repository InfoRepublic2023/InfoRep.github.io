Dynamic Array Formulas

Excel Know How v104.pptx

Formulas that can return arrays of variable size are called DYNAMIC ARRAY formulas.

Dynamic Array formulas will return an array, that automatically populates to neighbouring cells as required.

This behaviour is called SPILLING
